# Curso de GitHub

Este repositorio es parte del **Curso de GitHub** en [Platzi](https://platzi.com/). Aquí refuerzo conocimientos sobre Git y GitHub con el profesor **Freddy Vega**, CEO de Platzi.

## Descripción
El objetivo de este curso es aprender y mejorar el uso de Git y GitHub, herramientas fundamentales para el control de versiones en proyectos de desarrollo de software. A través de prácticas y ejemplos, se exploran conceptos clave como:

- Configuración inicial de Git
- Flujo de trabajo en Git
- Uso de ramas y `merge`
- Resolución de conflictos
- Trabajo con repositorios remotos
- Contribuciones y Pull Requests

## Contenido
Este repositorio incluye ejemplos y ejercicios prácticos abordados durante el curso, incluyendo:

- **Inicialización de repositorios**: creación de repositorios locales y conexión con repositorios remotos.
- **Manejo de ramas**: creación, fusión, y eliminación de ramas para un flujo de trabajo organizado.
- **Resolución de conflictos**: práctica en la resolución de conflictos de merge.
- **Buenas prácticas en commits**: mensajes claros y consistentes.
- **Contribuciones a proyectos**: cómo realizar Pull Requests y participar en proyectos de código abierto.
